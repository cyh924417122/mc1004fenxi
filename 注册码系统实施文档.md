# 注册码系统实施文档

## 1. 系统概述

本注册码系统用于生成和验证20位数字注册码，主要用于软件授权和产品激活。系统采用多层安全机制，包括伪随机映射、数据旋转和双重校验，确保注册码的安全性和可靠性。

## 2. 核心组件

### 2.1 字符集

系统使用两个字符集：

```python
A_ABCDEFGHJKMNPQRSTUVWXYZ = "ABCDEFGHJKMNPQRSTUVWXYZ"  # 23个可用字符（无I与O）
A_ACDEFGHJKLMNPQRSTUVWXYZ = "ACDEFGHJKLMNPQRSTUVWXYZ"  # 23个可用字符（无B/I/O）
```

### 2.2 核心函数

#### 2.2.1 计算产品ID的OEM哈希

```python
def compute_oem_hash(product_id):
    hash_val = 97
    while product_id:
        digit = product_id % 23
        product_id //= 23
        hash_val = 12157 * (digit + hash_val) % 12167
    return hash_val
```

**功能**：计算产品ID的OEM 23进制哈希值
**参数**：`product_id` - 产品ID
**返回值**：哈希值

#### 2.2.2 将哈希值编码为3个字母字符

```python
def encode_oem_chars(oem_value):
    hash_val = compute_oem_hash(oem_value)
    chars = []
    # 计算三个字符
    chars.append(A_ABCDEFGHJKMNPQRSTUVWXYZ[hash_val % 23])
    hash_val //= 23
    chars.append(A_ABCDEFGHJKMNPQRSTUVWXYZ[hash_val % 23])
    high_part = hash_val // 23
    chars.append(A_ABCDEFGHJKMNPQRSTUVWXYZ[high_part % 23])
    # 反转顺序，因为计算时是从低位到高位
    return ''.join(reversed(chars))
```

**功能**：将哈希值编码为3个字母字符
**参数**：`oem_value` - OEM值
**返回值**：3个字母组成的编码字符串

#### 2.2.3 生成0-9的映射表

```python
def generate_digit_permutation_map(input_seed, out_map):
    # 如果种子直接被模数整除，则用997避免全0序列
    if not (input_seed % 0x14DB3):
        input_seed = 997
    
    # 生成10个伪随机值（线性同余PRNG）
    rand_values = []
    for i in range(10):
        input_seed = 997 * input_seed % 0x14DB3
        rand_values.append(input_seed)
    
    # 初始化映射位置计数
    for i in range(10):
        out_map[i] = 0
    
    # 计算"排名"（0..9）: 值越小排名越小
    for i in range(10):
        for j in range(i + 1, 10):
            if rand_values[i] >= rand_values[j]:
                out_map[j] += 1
            else:
                out_map[i] += 1
    
    return input_seed
```

**功能**：生成0-9的伪随机映射表
**参数**：
- `input_seed` - 基于序列号的哈希种子值
- `out_map` - 10字节输出缓冲区
**返回值**：最终伪随机状态

#### 2.2.4 单字节滚动哈希

```python
def digit_hash_step(byte_value, current_hash):
    result = current_hash
    while byte_value:
        digit = byte_value % 10
        byte_value //= 10
        result = 991 * (digit + result) % 997
    return result
```

**功能**：对一个字节值按十进制拆位计算滚动哈希
**参数**：
- `byte_value` - 字节值
- `current_hash` - 当前哈希值
**返回值**：新的哈希值

#### 2.2.5 计算数据哈希

```python
def compute_data_hash(data_ptr, len):
    hash_val = 97  # 初始值 97
    for i in range(len):
        hash_val = digit_hash_step(data_ptr[i], hash_val)
    return hash_val
```

**功能**：计算字节序列的滚动哈希
**参数**：
- `data_ptr` - 数据指针
- `len` - 长度
**返回值**：哈希值

## 3. 注册码生成过程

### 3.1 输入参数

- `product_id`：产品ID，如15044657
- `registration_payload`：注册载荷，如"231251190900KQRX"

### 3.2 生成步骤

1. **解析注册载荷**：
   - 提取前12字节作为种子参数
   - 提取中间3字节的OEM编码字符
   - 提取最后1字节的尾部字符

2. **准备字段数据**：
   - 产品码：使用输入的product_id
   - 日期：使用系统日期或固定值
   - 尾数：使用固定值或随机值

3. **构建数据数组**：
   - 填充产品码（8位）
   - 填充日期（4位）
   - 填充尾数（4位）

4. **计算校验值**：
   - 对数据数组计算哈希值
   - 填充校验值（3位）

5. **计算校验和**：
   - 计算前19位数字的和
   - 对10取模得到偏移检验值

6. **数据旋转**：
   - 根据偏移检验值对数据进行旋转

7. **生成映射表**：
   - 计算种子哈希
   - 生成0-9的伪随机映射表

8. **正向映射**：
   - 将序号通过映射表转换为最终值
   - 添加偏移检验值作为第20位

9. **生成校验码**：
   - 将20位数字转换为字符串

### 3.3 代码实现

```python
def generate_20_digit_checksum(product_id, registration_payload):
    # 解析注册载荷
    src = registration_payload[:12]
    oem_chars = registration_payload[12:15]
    tail_char = registration_payload[15]
    
    # 计算种子参数
    seed_param = [int(c) for c in src] + [0] * 4  # 补全16字节
    
    # 生成字段数据
    product_code = product_id
    date_code = 260512  # 假设日期
    tail_code = 123  # 假设尾数
    
    # 构建v22数组
    v22 = [0] * 40
    
    # 填充产品码（8位）
    temp = product_code
    v22[0] = temp % 10
    temp //= 10
    v22[1] = temp % 10
    temp //= 10
    v22[2] = temp % 10
    temp //= 10
    v22[3] = temp % 10
    temp //= 10
    v22[4] = temp % 10
    temp //= 10
    v22[5] = temp % 10
    temp //= 10
    v22[6] = temp % 10
    temp //= 10
    v22[7] = temp % 10
    
    # 填充日期（4位）
    temp = date_code
    v22[8] = temp % 10
    temp //= 10
    v22[9] = temp % 10
    temp //= 10
    v22[10] = temp % 10
    temp //= 10
    v22[11] = temp % 10
    
    # 填充尾数（4位）
    temp = tail_code
    v22[12] = temp % 10
    temp //= 10
    v22[13] = temp % 10
    temp //= 10
    v22[14] = temp % 10
    temp //= 10
    v22[15] = temp % 10
    
    # 计算校验值
    v9 = compute_data_hash(v22, 16)
    
    # 填充校验值（3位）
    temp = v9
    v22[16] = temp % 10
    temp //= 10
    v22[17] = temp % 10
    temp //= 10
    v22[18] = temp % 10
    
    # 计算校验和
    v10 = sum(v22[:19])
    v5 = v10 % 10  # 偏移检验值
    
    # 旋转数据
    for i in range(19):
        v22[i + 19] = v22[i]
    
    # 执行旋转
    offset = 3 + v5
    for k in range(19):
        v22[(k + offset) % 19] = v22[19 + k]
    
    # 计算种子哈希
    v17 = compute_data_hash(seed_param, 16)
    
    # 生成数字映射表
    v23 = [0] * 16
    generate_digit_permutation_map(v17 + v5, v23)
    
    # 正向映射
    input_data = [0] * 20
    for i in range(19):
        input_data[i] = v23[v22[i]]
    
    # 添加偏移检验值
    input_data[19] = v5
    
    # 转换为字符串
    checksum = ''.join(map(str, input_data))
    
    return checksum
```

## 4. 注册码解码和验证过程

### 4.1 输入参数

- `input_data`：20位注册码数据
- `seed_param`：种子参数

### 4.2 验证步骤

1. **读取偏移检验值**：
   - 从输入数据的第20位读取偏移检验值

2. **计算种子哈希**：
   - 对种子参数计算哈希值

3. **生成映射表**：
   - 基于种子哈希和偏移检验值生成映射表

4. **逆向映射**：
   - 将输入数据的前19位通过映射表转换为序号

5. **数据旋转**：
   - 根据偏移检验值对数据进行旋转，还原原始顺序

6. **提取字段**：
   - 从旋转后的数据中提取产品码、日期、尾数和校验值

7. **计算校验哈希**：
   - 对旋转后的数据计算哈希值

8. **双重校验**：
   - 校验和检查：前19位数字之和对10取模应等于偏移检验值
   - 哈希校验：计算得到的哈希值应等于提取的校验值

9. **返回结果**：
   - 根据校验结果设置状态码
   - 返回解码后的数据

### 4.3 代码实现

```python
def decode_and_verify_registration_payload(output_buf, input_data, seed_param):
    # 初始化变量
    v13 = 0
    v14 = 0
    v15 = 0
    v16 = 0
    v22 = [0] * 40  # 40字节缓冲区
    v23 = [0] * 16  # 16字节映射表
    
    # 读取偏移检验值（最后一位）
    v5 = input_data[19]
    
    # 计算种子映射哈希
    v17 = compute_data_hash(seed_param, 16)
    
    # 生成数字映射表
    generate_digit_permutation_map(v17 + v5, v23)
    
    # 逆向映射
    for i in range(19):
        for j in range(10):
            if input_data[i] == v23[j]:
                v22[i] = j
                v22[i + 19] = j
                break
    
    # 旋转，还原数据顺序
    offset = 3 + v5
    for k in range(19):
        v22[k] = v22[k + offset]
    
    # 提取字段
    v18 = (10 * v22[1] + v22[0] + 100 * v22[2] + 1000 * v22[3] +
           10000 * v22[4] + 100000 * v22[5] + 1000000 * v22[6] + 10000000 * v22[7])
    
    v21 = 10 * v22[9] + v22[8] + 100 * v22[10] + 1000 * v22[11]
    v20 = 10 * v22[13] + v22[12] + 100 * v22[14] + 1000 * v22[15]
    v19 = 10 * v22[17] + v22[16] + 100 * v22[18]
    
    # 计算重新编码哈希
    v9 = compute_data_hash(v22, 16)
    
    # 校验 19 位和尾偏移
    v10 = 0
    for m in range(19):
        v10 += v22[m]
    
    if v10 % 10 == v5:
        if v9 == v19:
            v14 = v18
            v15 = v21
            v16 = v20
        else:
            v13 = 2  # 内部校验失败
        return copy_memory_optimized(output_buf, [v13, v14, v15, v16], 16)
    else:
        v13 = 1  # 校验和失败
        return copy_memory_optimized(output_buf, [v13, v14, v15, v16], 16)
```

## 5. 状态码说明

| 状态码 | 含义 |
|-------|------|
| 0 | 校验成功，所有验证都通过 |
| 1 | 校验和失败，前19位数字之和对10取模不等于第20位数字 |
| 2 | 内部校验失败，计算得到的哈希值与提取的校验值不匹配 |

## 8. 安全考虑

1. **伪随机映射**：使用线性同余PRNG生成随机映射表，增加破解难度
2. **数据旋转**：根据偏移检验值对数据进行旋转，进一步混淆数据
3. **双重校验**：同时使用校验和和哈希值进行验证，提高安全性
4. **多字段提取**：从注册码中提取多个字段，用于不同的验证目的

## 9. 总结

本注册码系统通过多层安全机制确保注册码的安全性和可靠性。生成过程包括字段填充、校验计算、数据旋转和映射转换，验证过程包括逆向映射、数据还原、字段提取和双重校验。系统设计合理，实现了完整的注册码生成和验证功能，可以有效防止未授权使用。
